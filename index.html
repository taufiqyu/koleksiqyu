Perjalanan Belajar Coding Saya

Saya memulai perjalanan belajar coding tanpa latar belakang IT. Berawal dari rasa ingin tahu, saya mulai belajar HTML, CSS, dan JavaScript melalui platform online seperti W3Schools dan Nimo.

Awalnya, semua terlihat rumit. Namun, saya terus mencoba dan belajar sedikit demi sedikit. Salah satu tantangan terbesar adalah memahami logika dasar coding dan bagaimana berbagai elemen bekerja sama untuk membangun sebuah website yang berfungsi dengan baik.

Saya memutuskan untuk membuat website undangan digital sebagai proyek pertama saya. Dari sini, saya belajar banyak hal, seperti:

  • Menggunakan Bootstrap untuk mendesain tampilan responsif.
  • Memanfaatkan JavaScript untuk membuat animasi dan fitur interaktif.
  • Hosting website di GitHub Pages dan menghubungkannya dengan domain yang saya beli.
  • Mengatasi masalah seperti error "Too Many Redirects" saat deployment.

Sekarang, website undangan digital saya sudah berjalan. Meski masih ada kekurangan, saya bangga karena semua ini hasil kerja keras dan dedikasi saya dalam belajar.

Bagi Anda yang juga sedang belajar coding, pesan saya adalah jangan takut mencoba. Gagal itu biasa, dan dari kegagalan itulah Anda akan belajar dan berkembang.

Terima kasih telah mendukung perjalanan saya ini!
